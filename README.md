# thunder-bot
kosen-13s' slackbot

## 実装済み機能一覧
### ping
`@thunder-bot ping`

`pong` と応答する

### ダイスロール
`[n]D[m]`

m面ダイスをn個振り、結果を表示する

### 大草原スロット
`大草原スロット`

スロットを表示し、reactionで止まる

### KVS
`/save-thunder-kvs [key] [value]`

`key` に `value` を保存する

`/load-thunder-kvs [key]`

`key` に保存された `value` を取得する

### 歴史に刻む
メッセージアクション `歴史に刻む`

メッセージの魚拓を取る
